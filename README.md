# Pytov
## Usage

Run using python (requires libraries: run `$ pip install -r requirements.txt`):
```sh
$ cd [folder_of_installation]
$ python pytov.py [path_to_your_pytov_code]
```
Run using executables:
```sh
$ cd [folder_of_installation]
$ cd exe
$ pytov [path_to_your_pytov_code]
```
